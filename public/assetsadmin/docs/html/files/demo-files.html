Demo Files

HTML/Handlebars

For Ace's demo pages I've used Handlebars templating library which makes it easy to separate large pages into smaller parts.
The files are located inside application/views/default .
The compiled HTML files are inside html folder.

Layouts

application/views/default/layouts folder contains layouts and their relevant partials (sections):

  • main.hbs is the main layout used in most pages

  • layout-2.hbs is used for Dashboard 2 page and it is different from main as navbar is inside content area (not above it)

  • layout-3.hbs is used for Dashboard 3 page and it is similar to main except it has different initial sidebar/navbar themes and boxed layout

  • login.hbs is used for the Login page

  • partials folder contains layouts sections such as sidebar, sidebar header, sidebar items, navbar, footer, settings box, etc ...

There is more information about layouts and using them in the relevant section

Pages

application/views/default/pages folder contains demo pages along with their partials which are different sections of a page.
For example for Profile page:

  • application/views/default/pages/page-profile.hbs is the main file
  • application/views/default/pages/partials/page-profile/ contains sections of the profile page such as timeline, activities, etc
  • application/views/default/pages/partials/page-profile/@page-script.js contains the script for this page, for example to initiate charts, etc
  • application/views/default/pages/partials/page-profile/@page-style.css contains the CSS used only in this page. I haven't included them in the main ace.css because they are not needed throughout Ace and are only used in this page
  • application/views/default/pages/partials/page-profile/@vendor-scripts.hbs contains the list of third party files and plugins used in this page

Data files

application/data folder contains data used in Ace's demo such as list of pages, navbar notifications, dashboard's stats, etc.

They are in json or csv format.

For example:

  • application/data/layouts/pages.json contains the page list (i.e. sidebar elements)
  • application/data/layouts/navbar-notif.json contains navbar notification items
  • application/data/pages/dashboard/conversations.json contains the conversation items displayed in dashboard

In Ace's demo pages, data is retrieved using a custom handlebars helper
{{#each (get "DATA_NAME")}}

Please note that you don't need these data files or the custom Handlebars helper in your app

hbs 
{{#each (get "conversations")}}
  <img src="{{avatar}}" />
  <div>{{message}}</div>
{{/each}}

{{#each (get "layout/navbar-notif")}}
   <div>{{title}}</div>
{{/each}}

Fonts & Assets

assets folder contains the following:

  • image which contains some images and avatars for demo
  • fonts which contains Open Sans font files used by default in Ace's demo. However you should load fonts directly from Google fonts. See Fonts & Icons
  • js folder which contains demo.js. See demo.js File
